# Course Prerequisites

To ensure everything runs smoothly during the course, please complete the steps below **before arriving**. This setup is essential so that all required software and packages are ready to use, allowing us to start the hands-on exercises immediately and make the best use of our time together.

The following steps will guide you through a few installations, in case you do not have the required software already, up to the creation and activation of the environment needed for the course. If you are unable to complete the environment activation successfully, please contact the organizers before the course starts.
